1. Présentation du projet

Secure Home est un système de domotique et de surveillance basé sur ESP32 et ESP32-CAM, intégrant des capteurs (laser, contact magnétique de porte) et une communication via Telegram.

This system is designed for educational and personal security automation purposes.

2. Objet de la politique

Cette politique décrit la manière dont les données sont collectées, utilisées et protégées dans le cadre du fonctionnement du système Secure Home.

This document explains how data is processed within the Secure Home ecosystem.

3. Définitions

  • ESP32 : microcontrôleur Wi-Fi utilisé pour l’automatisation.
  • ESP32-CAM : module caméra pour capture d’images.
  • Telegram : service de messagerie utilisé pour les notifications.
  • Données : informations transmises par les capteurs.

4. Données traitées

Données collectées

  • État des capteurs (porte, laser)
  • Images capturées par ESP32-CAM
  • Logs techniques (connexion, erreurs)

Données non collectées

  • Identité personnelle des utilisateurs
  • Données bancaires
  • Localisation GPS précise

Finalité

Les données servent uniquement à la surveillance locale et aux notifications de sécurité.

5. Sécurité informatique

  • Communication via HTTPS/TLS (Telegram API)
  • Protection du token Telegram
  • Réseau Wi-Fi sécurisé recommandé (WPA2/WPA3)
  • Accès limité aux dispositifs ESP32

Les images et données ne sont pas stockées sur des serveurs externes autres que Telegram.

6. Base juridique (RGPD)

Le traitement des données repose sur l’intérêt légitime de sécurité et d’automatisation domestique.

This project is compliant with GDPR principles where applicable.
